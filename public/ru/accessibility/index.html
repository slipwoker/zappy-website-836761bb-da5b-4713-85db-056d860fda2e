Декларация доступности

עדכון אחרון:19.1.2026

Израильский теннисный центр обязуется обеспечить доступность услуг для всех слоев населения. Мы работаем над внедрением рекомендаций израильского стандарта (ТИ 5568), основанного на рекомендациях WCAG 2.1 уровня AA.

רמת נגישות: AA (תקן ישראלי ת"י 5568)

1. Политика доступности

Мы верим в равенство прав и в право каждого человека получать информацию и услуги самостоятельно и на равных условиях. Этот сайт был разработан и создан с особым акцентом на доступность для людей с ограниченными возможностями.

2. Меры обеспечения доступности на сайте

  • Поддержка навигации с помощью клавиатуры
  • Поддержка программ чтения с экрана
  • Подходящая контрастность цветов
  • Гибкие размеры шрифтов с возможностью увеличения
  • Подробные описания изображений (Alt text)
  • Иерархически структурированные заголовки
  • Описательные и понятные ссылки
  • Поддержка масштабирования до 200% без потери контента

Соответствие стандарту WCAG 2.1

Сайт соответствует уровню AA стандарта WCAG 2.1, который включает:

  • Воспринимаемый (Perceivable): информация представлена способами, которые пользователи могут воспринять
  • Управляемый (Operable): компоненты интерфейса поддаются управлению
  • Понятный (Understandable): информация и пользовательский интерфейс понятны
  • Надежный (Robust): контент может быть интерпретирован различными технологиями

3. Поддерживаемые вспомогательные технологии

  • NVDA (Windows)
  • JAWS (Windows)
  • VoiceOver (Mac/iOS)
  • TalkBack (Android)
  • Dragon NaturallySpeaking
  • Switch navigation

4. Проверки доступности

Сайт проверяется с помощью:

  • Автоматические инструменты проверки доступности
  • Ручные проверки с программами чтения с экрана
  • Навигация только с помощью клавиатуры
  • Проверка контрастности цветов
  • Тестирование с пользователями с ограниченными возможностями

5. Постоянные улучшения

Мы постоянно работаем над улучшением доступности сайта и обновляем его в соответствии с технологическими разработками и отзывами пользователей.
