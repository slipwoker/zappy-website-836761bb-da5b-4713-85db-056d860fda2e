    • Accessibility Statement

    Last updated: 19.1.2026

    The Israeli Tennis Center is committed to making services accessible to the general public. We work to implement the guidelines of the Israeli Standard
    (IS 5568) based on WCAG 2.1 guidelines at AA level.

    Accessibility level: AA (Israeli Standard SI 5568)

    1. Accessibility Policy

    We believe in equal rights and in every person's right to receive information and services independently and equally.
    This website was designed and built with special emphasis on accessibility for people with disabilities.

    2. Website Accessibility Arrangements

    • Keyboard navigation support
    • Screen reader support
    • Appropriate color contrast
    • Flexible and scalable font sizes
    • Detailed image descriptions (Alt text)
    • Hierarchically structured headings
    • Descriptive and clear links
    • Zoom support up to 200% without content loss

    WCAG 2.1 Standard Compliance

    The website complies with AA level of WCAG 2.1 standard, which includes:

    • Perceivable: Information is presented in ways users can perceive
    • Operable: Interface components are operable
    • Understandable: Information and user interface are understandable
    • Robust: Content can be interpreted by a variety of technologies

    3. Supported Assistive Technologies

    • NVDA (Windows)
    • JAWS (Windows)
    • VoiceOver (Mac/iOS)
    • TalkBack (Android)
    • Dragon NaturallySpeaking
    • Switch navigation

    4. Accessibility Testing

    The website was tested using:

    • Automated accessibility testing tools
    • Manual testing with screen readers
    • Keyboard-only navigation
    • Color contrast testing
    • Testing with users with disabilities

    5. Continuous Improvements

    We work continuously to improve website accessibility and update it according to technological developments and user feedback.

    6. Contact and Feedback

    If you encountered an accessibility problem on the website or need assistance, please contact us:

    • Company: Israeli Tennis Center

    Response times: We respond to accessibility inquiries within 3 business days and work to resolve issues within 10 business days.
